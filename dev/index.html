MatrixFunctionDiff.jl

A julia package for computing Fréchet derivatives of scalar functions with matricies as variables. The higher order Fréchet derivatives are first written in a formula similar to the Daleskii-Krein theorem, and then computed by the divided difference.

Installation

MatrixFunctionDiff.jl is a registered package, so it can be installed by running

julia> Pkg.add("MatrixFunctionDiff")
